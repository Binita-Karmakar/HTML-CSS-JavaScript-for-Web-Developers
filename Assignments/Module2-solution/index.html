HTML,CSS & JS for web Developers

HTML

HTML stands for Hyper Text Markup Language. It is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as CSS & JavaScript to create web pages, as well as to create user interfaces for mobile and web applications. HTML describes the structure of a web page semantically & originally included cues for the appearance of the document.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. Along with HTML & JavaScript, CSS is a cornerstone technology of most websites to create visually engaging webpages, user interfaces for web applications & many mobile applications. It is designed to enable the separation of presentation & content, including layout, colors & fonts.

JS

JavaScript often abbreviated as JS, is a high-level, dynamic, untyped, & interpreted programming language that also a multi-paradigm language. Alongside HTML & CSS, JavaScript is one of the core technologies of the World Wide Web. It has application programming interfaces(APIs) for working with text, dates, regular expressions, standard data structures, & the Document Object Model(DOM).
